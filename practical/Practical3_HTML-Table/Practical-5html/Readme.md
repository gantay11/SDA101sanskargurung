# Hostel Maintenance Form Webpage

This project is a simple HTML-based form that allows students to submit hostel maintenance issues. It collects basic personal information, room details, and descriptions of the problem.

 Input fields for full name, email, student ID, and phone number
 Dropdown lists for block/building name, issue type, and emergency level
 Separate sections for personal details, room information, and issue description
 Textarea for detailed issue reporting
 
 How to Use

1. Save the file as `index.html`
2. Create a separate `style.css` file if needed
3. Open the HTML file in any browser
4. Fill out the form fields as required

 Future Improvements

* Add validation for required fields
* Fix label spelling errors
* Add a submit button and form handling
* Make the form responsive for mobile screens

 Purpose

This webpage was created to practice building structured HTML forms for real-world use cases such as hostel maintenance reporting.
