# Event Schedule Webpage

This project displays important event information using a simple and clean HTML structure with basic CSS styling. It includes details for night shows, tournaments, food menu timings, and welcome events.

 How to Use
1. Save the file with `.html` extension
2. Open it in any web browser
3. All event details will appear neatly formatted

 Future Improvements
Add navigation bar
Add images or icons for visual appeal
Make layout responsive for mobile devices
Improve styling with external CSS file

 ##Marks Table Webpage#

This project displays a student's academic performance using a structured HTML table styled with CSS. It includes module names, assignment titles, maximum marks, obtained marks, percentages, and grades.

How to Use

1. Save the file with `.html` extension
2. Open it in any browser
3. The marks table will display automatically
Future Improvements
 Add overall GPA calculation
 Make the design fully responsive
 Fix spelling errors (e.g., “PROBILITY” → “PROBABILITY”)
 Add a header and footer for better layout
The webpage was created to practice HTML tables and apply CSS to present student marks in a clear, attractive format.
