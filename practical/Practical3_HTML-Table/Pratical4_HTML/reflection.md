Short Reflection on the Event Schedule Webpage

This mini project helped me understand how to present event information clearly using simple HTML and CSS. By organizing details like time, location, and event type into headings and lists, I learned how structuring content properly improves readability. Styling the page using background color, borders, shadows, and fonts made the webpage more visually appealing, and I realized how small CSS properties can create a clean and attractive layout.

Through this activity, I also became more aware of consistency in spelling, formatting, and capitalization, which affects the overall professionalism of a webpage. This project strengthened my confidence in designing neat, organized web pages using basic HTML and CSS.


Short Reflection on Marks Table Webpage

Creating this marks table webpage helped me practice organizing academic data clearly using HTML tables. By working with table rows, headers, and columns, I learned how to structure information like module names, marks, and grades in a neat and readable format. Applying CSS styling allowed me to experiment with background colors, borders, hover effects, and alignment to make the table visually appealing. This activity improved my understanding of table formatting and reinforced how small design changes can make information easier to understand. Overall, the webpage enhanced my confidence in presenting structured data using HTML and CSS.
