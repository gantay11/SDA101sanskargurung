# Student Profile Webpage

This project is a simple HTML and CSS webpage that displays a student profile, including navigation links, personal details, timetable, daily activities, and study goals.


 A navigation bar with links such as Home, About, Contact, Logout, and Sign In
 Personal introduction and profile photo
 Weekly timetable displayed using an ordered list
 Daily routine activities displayed using an unordered list
 Study goals with nested lists
 Basic CSS styling for background, text color, layout, and hover effects

 How to Run

1. Download all files into one folder
2. Open **index.html** in any browser (Chrome, Firefox, Edge, etc.)
3. The webpage will load with the navigation bar and student profile content
