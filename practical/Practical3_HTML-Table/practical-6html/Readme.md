# Portfolio Website

This project is a simple, modern personal portfolio website designed using HTML and CSS. It showcases sections such as Home, About, Skills, Portfolio, and Contact.

 Features

 using Flexbox
 with name, role, and CTA button
 using Flexbox for skill cards
 using CSS Grid to display project thumbnails
 Clean and organized layout suitable for beginners

 How to Use

1. Save the file as `index.html`
2. Create a `style.css` file for styling
3. Open the HTML file in any browser
4. The website will load and display your portfolio sections

Future Improvements

* Add smooth scroll animations
* Add a real contact form
* Improve color theme and typography
* Replace placeholder images with real project screenshots

 Purpose

The project was created to practice Flexbox, Grid, and overall webpage layout while building a clean and functional portfolio website.
