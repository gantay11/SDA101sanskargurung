# Registration Form Validation

This project is a simple registration form that performs real-time validation on user inputs such as username, email, password, and confirm password. It is designed to help practice DOM manipulation, event handling, and client-side form validation.

 Features

 Username, email, password, and confirm password input fields
 Real-time validation using JavaScript
 Error messages displayed dynamically
 Submit button disabled until all fields are valid
 Organized and clean form structure

 Technologies Used

 for the form layout
 for styling and error message formatting
 for validation logic

 How to Use

1. Save the files as:

   * `index.html`
   * `style.css`
   * `script.js`
2. Open `index.html` in a browser
3. Enter data in each field
4. Fix errors based on the displayed messages
5. When all fields are valid, the **Register** button will activate

 Future Improvements

* Add regex-based email validation
* Improve password strength checking
* Add success message after form submission
* Make the form responsive for all devices

 Purpose

This project was created to practice building interactive forms and implementing client-side validation to improve user experience.
